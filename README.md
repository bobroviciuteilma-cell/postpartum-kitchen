## How it works
- `index.html` — the public, shareable, mobile-responsive page (also prints to PDF).
- `build.py` — generates the public `index.html` from a private master HTML, stripping personal
  medical details and renumbering. Run: `python3 build.py /path/to/master.html index.html`

Recipes from around the web are credited and linked to their original creators.

*General educational content, not medical advice — check anything health-related with your own provider.*
